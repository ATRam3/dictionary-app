# 📖 Dictionary App

A modern and responsive dictionary web application built with **React** that allows users to search for word definitions, pronunciations, meanings, synonyms, antonyms, and more using a Dictionary API.

---

# 📸 Preview


* **Live Demo:** `https://wordnest-peach.vercel.app/`
* **GitHub Repository:** `https://github.com/ATRam3/dictionary-app`

---

# ✨ Features

* 🔍 Search for any English word
* 📚 View multiple definitions
* 🔊 Listen to word pronunciation (if available)
* 📝 See parts of speech
* 💡 Example sentences
* 🔄 Synonyms and antonyms
* 🌙 Dark/Light mode
* 📱 Fully responsive design
* ⚡ Fast and lightweight interface
* ❌ Error handling for invalid words
* 🎨 Clean and modern UI

---

# 🛠️ Built With

* **React.js**
* **JavaScript (ES6+)**
* **CSS3**
* **Dictionary API**
* **Fetch API**

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ATRam3/dictionary-app.git
```

Go into the project folder:

```bash
cd dictionary-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```

---

# 🚀 Usage

1. Enter a word in the search box.
2. Press **Enter** or click the **Search** button.
3. View:

   * Definitions
   * Pronunciation
   * Parts of speech
   * Example sentences
   * Synonyms
   * Antonyms

# 🌐 API Used

This project uses the **Free Dictionary API**.

Example endpoint:

```
https://api.dictionaryapi.dev/api/v2/entries/en/<word>
```
